General Bot

Emergency Response Liberty County General Bot

Server Count: 2

This bot is written in Python.

Invite General Bot

Available Commands

  • $ban - Bans a user from the server.
  • $unban - Unbans a user from the server.
  • $mute - Mutes a user in the server.
  • $unmute - Unmutes a user in the server.
  • $lock - Locks a channel so that people cannot send messages.
  • $unlock - Unlocks a channel so that people can send messages again.
  • $say - Sends a message as the bot.
  • $version - Displays the bot's version number.
  • $info - Displays information about the server.
  • $purge - Purges a specified number of messages from a channel.
  • $role - Roles a specific person mentioned with a role mentioned.
  • $userinfo - Displays information about a user.
  • $flip - Flips a coin and displays the result.
  • $generate - Generates Random Account (Premium Command).
  • $generate-5char - Generates Random Account with 5 Characters (Premium Command).
  • $generate-offsale - Generates offsale, High Chance of not Working (Premium Command).
  • $generate-2017-2018 - Generates random 2017 or 2018 account.
  • $setup - Starts a VC only session vote.
